## How to Install

```
git clone https://github.com/Michael-Sebero/Numerology-Calculator

cd Numerology-Calculator && sh numerology-calculator
```

### Birthdate Calculator
This python script calculates your Life Path, Soul Urge, Expression, Personality, Physical Plane, Mental Plane, Emotional Plane and Intuitive Plane via Pythagorean Numerology and Planes of Expression. 

### Calendar Calculator
This Python script analyzes every day in a given month and year to calculate each date's Greek Pythagorean, Hebrew Gematria, and Combined Synthesis numbers, revealing the cosmological and spiritual significance of the entire month across two ancient numerological traditions.

### Date Calculator
This Python script analyzes any calendar date to calculate its Greek Pythagorean, Hebrew Gematria and Combined Synthesis numbers, revealing the cosmological and spiritual significance of that date across two ancient numerological traditions.

### Master Number Name Generator
This python script will generate names which equate to master number values.
